## 🛠️ 開發技術棧 (Tech Stack)
* **核心技術**：HTML、CSS、JavaScript、Bootstrap



### 💡 核心成果與描述
* **專案簡介**：
建立店家資訊與商品網頁內容，讓使用者可以了解店家的相關資訊、最新消息、商品資訊、聯繫方式等，還有模擬使用者訂購餐點的流程



* **成果描述**：
1. 網頁上方有標題分類，讓使用者點擊後可以進入該頁面
2. 影片、聯繫方式旁邊的圖片有做成超連結，點擊後可直接導入該頁面，方便使用者觀看與聯繫
3. 模擬使用者可以直接填寫表單與勾選餐點，但如要完成「訂購流程」，會直接帶入登入系統頁面，登入後才可結帳
